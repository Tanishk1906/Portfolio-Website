# 👩‍💻 Tanishka Meena | Personal Portfolio Website 🌐

Welcome to the repository of my personal portfolio website! This dynamic and fully responsive website was built to showcase my academic journey, technical skills, featured projects, and achievements. 

This project was developed as part of **Assessment Task 1** for the **InAmigos Foundation**.

## 🚀 Live Demo


## ✨ Key Features
*   **Modern UI/UX:** Clean, glassmorphism aesthetic with a smooth, professional layout.
*   **Dark/Light Mode:** Integrated theme toggler for optimal user experience.
*   **Interactive Animations:** Features a custom particle background canvas, scroll-reveal animations, and dynamic typing text.
*   **Fully Responsive:** Perfectly optimized for desktops, tablets, and mobile devices.
*   **Clean Architecture:** Code is modularized into separate HTML, CSS, and JavaScript files for high maintainability.

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure and layout.
*   **CSS3:** Custom CSS variables, Flexbox, CSS Grid, and keyframe animations (No external CSS frameworks used).
*   **JavaScript (ES6+):** DOM manipulation, event listeners, intersection observers for scroll effects, and HTML5 Canvas API for particle animations.
